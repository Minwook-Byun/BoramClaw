BoramClaw 시스템 아키텍처 도식, 깔끔한 흰 배경, 모던 플랫 인포그래픽.

제목: "BoramClaw Autonomous Agent"

필수 블록:
1) 사용자 인터페이스: CLI / Web UI / Telegram
2) Gateway + ReAct 루프: Thought -> Action -> Observation
3) Tool Router + Dynamic Tool Loader (tools/*.py)
4) Tool Sandbox: 파일시스템 경계, 네트워크 정책
5) Scheduler + Heartbeat
6) Watchdog 4-Tier: KeepAlive, Watchdog, Guardian, Emergency LLM Recovery
7) Memory: 단기 메모리 + 벡터 장기 메모리
8) Logs / Metrics / Audit
9) 외부 서비스: arXiv, Gmail, Calendar, GitHub, On-chain wallet APIs
